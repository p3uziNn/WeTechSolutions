# 💼 WeTechSolutions

Empresa focada na venda de sistemas e planilhas automatizadas, com geração dinâmica de páginas de compra utilizando Python.

Projeto desenvolvido com foco em automação, escalabilidade e profissionalização do processo de vendas digitais.

---

## 📌 Sobre o Projeto

A **WeTechSolutions** foi criada com o objetivo de oferecer sistemas e planilhas automatizadas para facilitar processos empresariais.

Ao invés de criar uma página estática para cada produto, o sistema foi desenvolvido para gerar páginas de venda dinamicamente utilizando Python, tornando o processo mais profissional, organizado e escalável.

---

## 🚀 Funcionalidades

- ✅ Listagem dinâmica de produtos
- ✅ Geração automática de páginas de venda
- ✅ Estrutura escalável (sem necessidade de criar uma página manual por produto)
- ✅ Organização modular do código
- ✅ Simulação de fluxo de compra

---

- Estrutura dinâmica de rotas/páginas
